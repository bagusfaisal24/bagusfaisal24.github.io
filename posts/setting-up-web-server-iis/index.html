Setting Up Web Server Internet Information Services (IIS) on Windows Server

March 4, 2021 One minute read
dev • devOps
devOps

1. Overview

In this tutorial we’re going to share how to setting up Web Server Internet Information Services (IIS) on windows server for deployment node js application. IIS is a web server feature on windows server.

2. Install the Web Server(IIS) in Server Manager Dashboard

3. Create new sites

Right click Sites and then setting up, follow step bellow.

4. Open your sites

Open your website with url / ip address setting up before. My site bindings to IP Address 192.168.55.71:8080

5. Enable web platform installer

Before connect with backend application, you must enable web platform installer and install plugin application request routing 3.0.

6. Connect With Backend Server Proxy

Right click on your Sites name in IIS dashboard manager and click new Application

7. Setting URL Rewrite rule

Click URL Rewrite -> Add rule(s) -> reverse proxy and setting your backend url / ip address location. I have backend application running in IP address 192.168.55.71:8081

8. Testing your backend connect with frontend application
