Cara Setup Instalasi Ngrok di Windows

September 4, 2020 2 minutes read
dev • ngrok • public url
webhook • ngrok

Ngrok adalah platform webhook public url, kegunaan ngrok salah satunya yakni expose local webserver anda agar bisa di akses publik tanpa hosting ataupun memakai vps.

kali ini saya akan memberikan cara setup instalasi ngrok di local agar website anda bisa di akses. Gambaran arsitektur cara kerja ngrok adalah seperti dibawah ini.

https://in.pinterest.com/pin/685321268270571841/

dari gambar diatas ngrok dapat diakses client melalui internet dengan menghiraukan firewall, karena tanpa melalui firewall sangat tidak disarankan untuk menggunakan ngrok pada environment production aplikasi yang berskala besar karena sangat beresiko jika website kita terbobol ini pendapat pribadi penulis.

baik langsung ke intinya saya akan memberikan sedikit tutorial cara untuk instalasi ngrok.

  • Langkah pertama download terlebih dahulu aplikasi ngrok. ngrok bisa digunakan di operation system windows, linux, ataupun macOs. berikut link untuk download aplikasinya https://ngrok.com/download

  • Setelah anda download langkah selanjutnya yakni extract/ unzip hasil download tersebut.

  • Connect auth token, sebelum menggunakan ngrok anda harus registrasi dulu di website ngrok https://dashboard.ngrok.com/login setelah anda registrasi kemudian connect dengan cara membuka terminal / cmd dan masuk ke direktori download unzip aplikasi ngrok ./ngrok authtoken <your_auth_token>

  • Langkah selanjutnya yakni buka hasil generate file ngrok.yml kalau di windows ada di direktori C://Users/nama user/.ngrok2

  • Menambahkan konfigurasi pada file ngrok.yml

  • Jalankan aplikasi ngrok, masuk ke direktori aplikasi ngrok hasil download sebelumnya. Untuk menjalankan aplikasi ngrok yakni dengan buka terminal / cmd ./ngrok.exe start --all

  • Setelah ngrok berjalan kemudian buka dan login ke website ngrok, buka menu status -> tunnels. Jika ngrok sukses berjalan maka akan muncul url yang digunakan public url aplikasi server.

Diatas adalah sedikit tutorial cara install ngrok di windows, terimakasih sudah membaca apabila ada kekurangan saya mohon maaf. happy learning:))
