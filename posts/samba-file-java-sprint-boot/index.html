Push File With Samba Client Java

February 22, 2020 2 minutes read
dev
smb file

Hallo hallo whaatsup kali ini saya coba bagikan sedikit ilmu dari case yang saya kerjakan beberapa waktu lalu, yakni push file ke Samba server dengan samba client.

Beberapa comporate dalam menjalankan bisnis nya ada yang menggunakan share folder, dimana share folder ini berguna apabila user ingin menggunakan komputer yang berbeda. Share folder ini bersifat private karena berisi file-file penting apabila ingin masuk membutuhkan authentikasi.

Nah case yang beberapa waktu lalu saya kerjakan yakni membuat program push file-file report ke share folder ke masing departement departement, jadi user di department tersebut tidak perlu download file lewat sistem tapi tinggal masuk share folder yang dimiliki.

Push file ini saya menggunakan Samba client, dan springboot program utamanya. Baik berikut beberapa cara implementasi nya:

Import library samba

Jika base project anda menggunakan gradle langsung saja pakai lib ini

compile('org.springframework.integration:spring-integration-smb:1.1.0.RELEASE')

setelah import langsung ke langkah selanjutnya.

Membuat Fungsi Util Write File

Pertama membuat fungsi authentikasi smb agar program push file kita dapat berjalan.

public SmbFile getaAthentication(String filePath, String username, String password, String domain) throws MalformedURLException {
    NtlmPasswordAuthentication authentication = new NtlmPasswordAuthentication(domain, username, password);
    return new SmbFile(filePath, authentication);
}

keterangan param variable:

  • filePath : Yakni path file yang akan kita push, contohnya seperti smb://localhost:8080/share-folder-example
  • username : username ini authentikasi samba client untuk bisa masuk ke share folder.
  • password : password ini authentikasi samba client untuk bisa masuk ke share folder.
  • domain : domain ini digunakan jika dalam config samba server ada.

Kedua membuat fungsi write file

public void write(byte[] data, SmbFile smbFile) throws IOException {
    SmbFileOutputStream smbFileOutputStream = new SmbFileOutputStream(smbFile);
    smbFileOutputStream.write(data);
    smbFileOutputStream.close();
}

keterangan param variable:

  • byte[] data : variable ini merupakan type data byte[] inilah byte[] data atau file yang akan kita push
  • smbFile : variable ini merupakan authentikasi agar file yang kita push bisa masuk ke share folder.

Jadi alur yang berjalan saat push file ke samba fungsi yang berjalan duluan getaAthentication dan jalankan fungsi write.
